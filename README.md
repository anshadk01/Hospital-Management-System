# ✨ Overview

The **Hospital Management System** is a modern healthcare web application designed to simplify and automate hospital workflows.

This platform provides:

- 🩺 Patient Appointment Booking
- 👨‍⚕️ Doctor Management
- 🛡️ Secure Authentication & Authorization
- 📊 Admin Dashboard
- 📁 Medical Record Handling
- 💳 Billing & Appointment Tracking
- 📱 Fully Responsive UI
- ⚡ Fast API Communication
- 🔐 JWT Protected Routes

The project follows a scalable MERN stack architecture with separate frontend, admin dashboard, and backend services.

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/Hospital-Management-System.git
```

---

## 2️⃣ Navigate into Project

```bash
cd Hospital-Management-System
```

---

# 📦 Install Dependencies

## Frontend

```bash
cd Frontend
npm install
```

## Admin Dashboard

```bash
cd ../AdminDash
npm install
```

## Backend

```bash
cd ../Backend
npm install
```

---

# 🔑 Environment Variables

Create a `.env` file inside the Backend folder.

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key
```

---

# ▶️ Run the Project

## Start Backend

```bash
cd Backend
npm run dev
```

## Start Frontend

```bash
cd Frontend
npm run dev
```

## Start Admin Dashboard

```bash
cd AdminDash
npm run dev
```
